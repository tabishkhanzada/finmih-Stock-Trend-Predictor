📈 finmih Stock Trend Predictor (AI + Streamlit)

An AI-powered web application that predicts stock price trends using deep learning. This project uses LSTM (Long Short-Term Memory) models to analyze historical stock data and visualize future trends in an interactive dashboard built with Streamlit.

🚀 Features
📊 Real-time stock data visualization
🤖 Deep learning model (LSTM) for prediction
📈 Trend forecasting based on historical data
🖥️ Interactive UI using Streamlit
⚡ Fast and lightweight deployment
📉 Moving averages (MA50, MA100, MA200) analysis
🧠 Tech Stack
Frontend/UI: Streamlit
Backend: Python
Machine Learning: TensorFlow, Keras
Data Processing: Pandas, NumPy
Visualization: Matplotlib

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/finmih-Stock-Trend-Predictor.git
cd finmih-Stock-Trend-Predictor
2️⃣ Create virtual environment (recommended)
python -m venv venv
venv\Scripts\activate   # Windows
3️⃣ Install dependencies
pip install -r requirements.txt
4️⃣ Run the app
streamlit run app.py

📊 How It Works
User enters a stock ticker (e.g., AAPL, TSLA)
App fetches historical stock data
Data is preprocessed and scaled
LSTM model predicts future trends
Results are displayed using graphs
